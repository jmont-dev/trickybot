# Trickybot

The bot that does everything. Programmed with an array of fun features for discord chat rooms.

Includes the following:

* Music and Youtube Playing
* Conversational AI and text generation (GPT-2/transformers)
* Wordcloud and QR code generation
* Wikipedia, Google, and Genius lookup
* And much more

Install dependencies:

```
pip install -r requirements.txt --user
```

Set the trickytoken environment before using:

```
set trickytoken=your_token_here
```
